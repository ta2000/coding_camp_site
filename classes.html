Introduction to Programming and Web Development
Day 1: Make a webpage with HTML.
Day 2: Learn about JavaScript variables, functions, loops, and objects.
Day 3: Use JavaScript to change and create HTML elements, and learn about event listeners.
Day 4: Make a webpage with HTML
Learn Game Mechanics & Build Prototype
Day 1: Create the Game object and start using the HTML canvas element.
Day 2: Learn about the game loop, and how to draw shapes, text, or an image on the canvas.
Day 3: Use objects, classes, and inheritance to add a controllable character and basic enemies to the game.
Day 4: Add a background image, and some extra logic to make the prototype look better.
