Classes
These classes will teach you how to make a 2D game. If you are taking a class without taking the classes before it, please make sure you know the material from those classes. The classes are listed here in the order they should be taken. Please bring your own laptop.
Intro to Programming/Web Development:
Session 1: 7/11 - 7/15
Session 2: 8/8 - 8/12
Game Mechanics and Prototype:
Session 1: 7/18 - 7/22
Session 2: 8/15 - 8/19
Build A Complete Web Game:
Session 1: 7/25 - 7/29
Session 2: 8/22 - 8/26
Introduction to Programming and Web Development
  • Make a basic webpage with HTML, with images, text, links, and styling.
  • Learn about JavaScript variables, functions, loops, objects, control flow, and logical operators.
  • Use JavaScript to change and create HTML elements.
  • Learn how to use event listeners.
    Learn Game Mechanics & Build Prototype
  • Create the Game object and start using the HTML canvas element. (The scene for the game.)
  • Learn about the game loop, and how to draw shapes, text, or an image on the canvas.
  • Use objects, classes, and inheritance to add a controllable character and basic enemies to the game.
  • Add a background image, and some extra logic to make the prototype look better.
    Build A Complete Web Game
  • Create a collection to hold all the game objects in, and create new objects during gameplay.
  • Make the game world free-roaming, and add background textures.
  • Create a dynamic user interface, to give the player information about their current state (Health bar, money, etc.).
  • Add mouse controls and projectiles.
  • Implement music and sound, and the ability to change them based on distance or location.
